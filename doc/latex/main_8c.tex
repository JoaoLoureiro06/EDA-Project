Função principal do programa. 

Esta função exibe um menu de opções para o usuário, permitindo a interação com o sistema de antenas. O programa permite carregar antenas de um arquivo, inserir novas antenas, remover antenas, listar antenas, deduzir localizações com efeito nefasto, e mostrar o mapa de antenas. O loop continua até que o usuário escolha a opção de sair.

Retorna 0 se o programa for executado com sucesso.
